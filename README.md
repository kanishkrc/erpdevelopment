Laboratory Management Software / ERP Development

This project is a full-stack Laboratory Resource and Workflow Management System developed at the Information Systems Lab, IIT Kharagpur. It digitizes and streamlines laboratory operations including equipment booking, approval workflows, user role management, and secure payments. The system improves efficiency, transparency, and accountability in lab operations.

Key Features

Secure CRUD APIs

Built scalable RESTful services using Node.js and Express for reliable lab resource management.

Role-Based Authentication

Implemented secure access control using Passport.js, sessions, and cookies for Technician, CSC, and User roles.

Online Payment System

Integrated Razorpay for seamless and automated payment processing.

Optimized Database Architecture

Designed optimized MongoDB schemas using Mongoose, with validation and hooks for data integrity.

Modern Front-End Interface

Developed responsive UI using React.js and Tailwind CSS, with reusable components and modern React APIs.
